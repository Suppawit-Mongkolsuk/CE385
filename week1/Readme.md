## สรุปไฟล์ workshop

ไฟล์ที่รวมอยู่ในที่นี้:

- `workshop1.1.js` — แสดงข้อมูลผู้ใช้แบบตรงไปตรงมา (ชื่อ, อายุ, สาขา)

  - ผลลัพธ์ที่คาดหวัง: พิมพ์ 3 บรรทัด เช่น "ชื่อ: ...", "อายุ: ...", "สาขา: ..."

- `workshop1.2.js` — ประกาศตัวแปรหลายชนิด (string, number, boolean, null, undefined) แล้วใช้ `typeof` แสดงชนิดข้อมูล

  - ผลลัพธ์ที่คาดหวัง: พิมพ์ชนิดของแต่ละตัวแปรทีละบรรทัด

- `workshop1.3.js` — คำนวณผลรวมและค่าเฉลี่ยของคะแนน 3 วิชา แล้วเช็คว่าเฉลี่ย >= 80 หรือไม่

  - ผลลัพธ์ที่คาดหวัง: พิมพ์คะเเนนรวม, คะเเนนเฉลี่ย และบรรทัดที่บอก `true` หรือ `false` ตามเงื่อนไข

- `workshop1.4.js` — ตัวอย่างเงื่อนไขการล็อกอินเบื้องต้น (ตรวจอายุก่อนแล้วตรวจ Username/Password)
  - หมายเหตุ: ในไฟล์มีค่า Username/Password เรียงไว้เป็นค่าตายตัว ถ้าต้องการให้ล็อกอินสำเร็จ ให้เปลี่ยน `Password` เป็น `"suppawit1410"` และตรวจสอบว่า `age` มากกว่า 18
